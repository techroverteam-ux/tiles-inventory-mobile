## 🚀 Installation Instructions

### Android Installation
1. **Transfer APK:** Copy `android/app/build/outputs/apk/debug/app-debug.apk` to your Android device
2. **Enable Unknown Sources:** Go to Settings > Security > Enable "Unknown Sources"
3. **Install:** Tap the APK file and follow installation prompts
4. **Launch:** Find "Tiles Inventory" app in your app drawer

### iOS Build (if needed)
```bash
# Navigate to iOS directory
cd ios

# Install CocoaPods dependencies
pod install

# Build for iOS (requires Xcode)
cd ..
npx react-native run-ios
```
